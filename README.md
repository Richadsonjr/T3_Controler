<h1 align="center">T3 Controler</h1>
<h3 align="center">Sistema de gerenciamento financeiro Com IA</h3>

<h3 align="left">Para utilizar siga os passos:</h3>
<p align="left">
</p>
1 - Certifique - se de ter instalado o nodejs em seu computador, caso contrario, baixe e instale o Node.js a partir do site oficial: https://nodejs.org/
2 - Navegue até a pasta srv pelo CMD e execute o comando:
    npm install
3 - Feche o CMD e na pasta raiz, execute o arquivo PlayApplication.bat  para iniciar a aplicação. 
4 - Acesse pelo navegador o endereço http://localhost:3000/ para acessar a aplicação.
5 - Mantenha as telas do CMD abertas caso contrario o sistema irá parar de funcionar.
